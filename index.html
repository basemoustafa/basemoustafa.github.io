Version Control System

A version control system in Java that offers a comprehensive set of 13 commands to manage code repositories effectively. The system includes essential operations such as initializing a repository, adding files, committing changes, merging branches, creating new branches, and switching between them.

To ensure the program's persistence and support local file operations, I designed a set of classes representing the internal structures during execution. These classes mirror the parallel representation of the code repository as local files.

Encypted File Sharing System

An encrypted file sharing system using Golang that prioritizes data security through the implementation of robust cryptographic techniques. The system ensures end-to-end encryption and prevents unauthorized access or tampering of shared files.

To achieve this, I incorporated cryptographic primitives like AES (Advanced Encryption Standard) and HMAC (Hash-based Message Authentication Code). These algorithms provide strong encryption and integrity checks, guaranteeing the confidentiality and integrity of the shared data.

The system offers user registration, login, and multiple session management features, allowing seamless sharing, receiving, and revoking of files even when users are offline. By leveraging RSA (Rivest-Shamir-Adleman) encryption, Argon2 password hashing, and HMAC, the system establishes a high level of security, rendering it computationally infeasible for attackers to recover information, modify files undetected, or impersonate users with control over the storage server.

Relational Database

A relational database system in Java that offers efficient and robust storage, retrieval, and manipulation of data using a B+ tree data structure.

The B+ tree data structure serves as the key-value storage mechanism for pages, enabling seamless addition, deletion, search, update, and iteration of entries. This structure optimizes performance by providing efficient access and retrieval operations.

Additionally, I implemented three join algorithms, namely page nested loop join, block nested loop join, and grace hash join, to facilitate efficient data retrieval from multiple tables. The system also incorporates query optimization techniques such as cost estimation, single table access selection, and join selection to enhance query performance.

To ensure concurrent access to the database, I developed a lock manager capable of handling multi-threaded concurrent page access and preventing potential deadlocks. This concurrency control mechanism ensures efficient and reliable execution of multiple operations in a high-performance database system.

Enigma Machine

A Java-based simulator of the Enigma machine, a famous encryption device used by the Germans during World War II. The simulator replicates the functionality of the Enigma machine and can handle various initial configurations and messages for encoding and decoding.

By creating a generalized simulator, I ensured the versatility of the program, allowing it to handle a wide range of possible configurations for the Enigma machine. This flexibility enables users to experiment with different settings and understand the encryption process in greater detail.

In the implementation, I leveraged Java's powerful data structures, such as String, HashMap, ArrayList, and Scanner, to handle string manipulation, data mapping, and file reading operations necessary for encryption and decryption.

AI Checkerboard Game

A Java-based recreation of Ataxx, a classic two-player checkers game, along with a graphical user interface (GUI) that allows players to choose between playing against a human opponent or a computer opponent.

The key highlight of this project is the implementation of an intelligent AI player for the computer opponent. The AI player utilizes game trees and alpha-beta pruning techniques to make strategic decisions based on heuristic values. This approach enables the computer opponent to analyze potential moves, predict outcomes, and choose the most advantageous move to play against the human player.

By creating this AI-based checkers game, I showcased my proficiency in Java programming and demonstrated my ability to implement complex game logic and intelligent decision-making algorithms. The project's GUI enhances the user experience by providing an interactive and visually appealing interface for playing Ataxx.
